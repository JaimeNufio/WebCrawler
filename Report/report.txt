Web Crawler Implentation:

  The web crawler's implementation leans heavily on the provided pseudocode, but diverges from it in a few key ways. Firstly, instead of saving the websites as they are deemed "good", the website's url is simply added to the list. Afterwards, once 500 websites are traversed and deemed good to use, an array of URL's is stepped through, and BeautifulSoup is used again to read create a file per each URL named after its title and position found, and then stores it in individual files, but also a master file with all the links data concatinated. Inputs are read from an easily readable JSON file. The way I've written it, .  Queue class is borrowed from Python, regex is used for searching strings. There isn't a terrible amount to say besides that when run, it lists all the websites as they are added to the saved list, and once done, it'll tell the user that it's begun saving.


The topic I chose was WWII, and anything related to the war.
